> 打仗這件事，本質都是「搶地盤 vs 燒錢」。

**MR（邊際收益）**：打贏了，多搶一塊地、多收一份稅、多一個官位。
**MC（邊際成本）**：死人、燒錢、得罪人、後勤崩了。

---

## 一、先說清楚：家康和崇禎面對的是同一筆生意

**戰國大名打仗**：
- MR = 搶隔壁大名10萬石土地，今年多收3萬石糧
- MC = 死500個足輕，燒掉5萬石糧草，修城再花2萬
- 打不打？算 MR - MC。如果是正的，就打。

**崇禎打仗**：
- MR = 剿滅李自成，保住河南，一年多收50萬兩稅
- MC = 派左良玉去打，軍餉300萬兩，徵稅逼反3萬新流寇
- 打不打？同樣算 MR - MC。

家康和崇禎是同行。區別在於：**家康後來修改了遊戲規則，讓全日本「打仗MR-MC永遠是負數」。崇禎沒改成，被遊戲規則玩死**。

---

## 二、史實：家康上位後，做了4件事，把「打仗」這門生意的MR幹廢、MC拉爆

1600年關原之戰後，家康拿下天下。但全國260個大名手裡還有刀。他要解決的問題就是：**怎麼讓所有大名算完賬，發現「打仗不如種地」？**

**第1件：參勤交代——把大名的MC直接拉滿**

規定大名一年住江戶，一年回領地。路費、江戶豪宅、老婆孩子當人質，全自費。加賀100萬石大名，一年收入100萬石，參勤交代就燒掉 **30—50萬石**。

結果：打仗的MC本來是60萬，日常開銷就先燒掉50萬。**MC佔收入95%，打仗等於破產**。

**第2件：改易轉封——把大名的MR打成負數**

沒兒子就沒收領地；犯錯就從100萬石貶到1萬石。260年間改易轉封超過 **200次**，外樣大名平均50年被搞一次。

結果：你打仗贏了，MR本來是多10萬石。但德川一句話就能沒收你100萬石。**MR = -100萬石，打仗不如不打**。

**第3件：武家諸法度・一國一城令——把啟動成本拉爆**

一個大名只准有一個城，其他全拆。武士不准住鄉下，全住城下町。

結果：你想造反，連個據點都沒有。**打仗的固定成本MC變得極高，還沒開打就先破產**。

**第4件：鎖國——把「請外援」這個MR刪了**

1639年鎖國，只留長崎。禁天主教，大名不准買西洋大砲。

結果：戰國時能找葡萄牙買火槍，MR很高。現在只能用刀跟德川的鐵砲部隊打，**MR = 送死**。

**四件做完，日本賬本變成**：

打仗MR = 0或負無窮。打仗MC = 100%收入 + 滅族。**MR - MC = -∞**。於是260年沒人打了。

---

## 三、對比崇禎：他面對同一道題，但MR和MC全反了

**打流寇：MR是負的，MC還巨高**

- MR = 打下河南，能收稅？河南大旱，人相食，**MR = -100萬兩**，還要賑災
- MC = 派兵300萬兩軍餉 + 徵稅逼反3萬農民，**MC = 900萬兩**
- **MR - MC = -1,000萬/年**。崇禎打了17年

**打後金：MR是0，MC是800萬**

- MR = 守住寧遠，後金不給你交1兩稅，**MR = 0**
- MC = 遼餉520萬 + 後勤戰事雜費280萬 = **800萬/年**
- **MR - MC = -800萬/年**。崇禎守到死

**家康如果穿越成崇禎，會怎麼改？**

1. **跟後金「參勤交代」**：不打了，一年給200萬歲幣，讓皇太極去瀋陽住。MC從800萬壓到200萬
2. **對流寇「改易轉封」**：招安李自成，給他河南，條件是打張獻忠。MR從-100萬翻成+300萬

但崇禎一條都沒做，因為「天子守國門」、「賊不可用」。他維護了面子，**MR - MC 繼續 -1,800萬/年**。

---

## 四、收束：為什麼家康能改規則，崇禎不能？

**家康的賬本**：
- 面子 = 0分：給天皇磕頭，給外樣大名叫哥，鎖國被罵膽小
- 里子 = 100分：MC趨近0，MR穩定，德川吃260年
- **結論：面子虧了，江山賺了**

**崇禎的賬本**：
- 面子 = 100分：不割地，不議和，不招安，不借債
- 里子 = -100分：國庫23萬兩，煤山上吊
- **結論：面子拉滿，江山拉沒**

同一道MR-MC題，答案不同，是因為CEO的KPI不同。

家康的KPI：**活下去**。所以他把「打仗」做成全日本最虧的生意。
崇禎的KPI：**當聖君**。所以他把「投降止損」做成全大明最虧的生意。
