前三篇把台灣當成封閉孤島討論——怎麼變富、怎麼解決失業、錢為什麼變薄。但台灣的出口佔 GDP 超過 70%，晶片賣美國、機台跟日本買、外匯存底全球前十。**只要跟外國有互動，總體經濟就多出兩條邏輯線。**

---

## 兩個基本流動：東西 vs. 錢

一個國家跟外國的互動只有兩種：**買賣東西**，以及**買賣資產**。

**淨出口 NX** = 出口 - 進口  
台灣賣晶片、買石油，出口大於進口就是貿易順差（NX > 0）。

**資本淨流出 NCO** = 台灣人買外國資產 - 外國人買台灣資產  
台積電去美國設廠、你買美股，是錢流出去（NCO > 0）；外資買台積電股票，是錢流進來（NCO < 0）。

**恆等式：NX = NCO**

你賣晶片賺了美金，這筆美金只有兩個去處：
1. 拿去買美國的石油或 iPhone → 變成進口，抵銷了出口的 NX
2. 拿去買美國的股票或美債 → 變成 NCO，對外投資

**賺到的外匯，最後不是消費掉，就是投資出去。** 沒有第三條路。

這就是為什麼貿易順差的國家，必然同時在對外投資。台灣每年順差，正是把賺到的美金買成美債或海外廠房。美國長期逆差，是因為全世界用貿易剩餘去買它的股票和國債。沒有誰佔誰便宜，這是會計恆等式。

---

## 匯率：名目 vs. 實質

你去銀行看到的「1 美元 = 32 台幣」，是名目匯率。

**實質匯率**更重要——它衡量的是「相同購買力下，在台灣 vs. 在美國能買多少東西」：

$$\text{實質匯率} = \text{名目匯率} \times \frac{\text{國內物價}}{\text{國外物價}}$$

台灣便當 100 台幣、美國便當 10 美元、名目匯率 32：

$$\text{實質匯率} = 32 \times \frac{100}{10} = 320$$

意思是：花在美國買一個便當的錢，拿來台灣可以買 3.2 個。**實質匯率越低，台灣的東西相對越便宜，出口競爭力越強，NX 越大。**

**購買力平價（PPP）的長期錨點：**

長期來看，同一件商品在各國的實質價格會趨於一致，否則就有套利空間。大麥克在台灣 80 台幣、美國 5 美元，那長期匯率就會往 80 ÷ 5 = 16 靠攏。現實中有關稅與運費，不會完全相等，但 PPP 是判斷一國貨幣是否被高估或低估的基準。

## 雙市場模型：可貸資金 + 外匯

開放後，台灣的儲蓄 S 有兩個去處：國內投資 I，或對外投資 NCO。

$$S = I + NCO$$

**NCO 取決於利率差：**
- 台灣利率 > 世界水準 → 外資湧入 → NCO 下降（甚至轉負）
- 台灣利率 < 世界水準 → 資金外流 → NCO 增加

兩個市場連動的機制：

```
可貸資金市場決定實質利率
→ 實質利率決定 NCO（資本要流入還是流出）
→ NCO 決定外匯市場的台幣供需
→ 供需決定實質匯率
→ 實質匯率決定 NX
```

---

## 用模型解釋三個政策場景

**場景一：政府預算赤字**

政府借錢 → 可貸資金需求增加 → 國內實質利率上升 → 外資流入（NCO↓）→ 台幣升值 → 出口變貴、進口變便宜 → NX 下降

> **結論：財政赤字帶來貿易逆差，這叫「雙赤字」。美國是教科書案例。**

**場景二：對特定國家課關稅**

直覺認為：進口少了，NX 應該增加。

但匯率會反應：進口減少 → 台幣不用換出去 → 台幣供給減少 → 台幣升值 → 出口跟著變貴 → NX 被壓回原點

> **結論：貿易政策無法改變貿易餘額，只能改變進出口的組成。** 貿易餘額由 S - I 決定，只要儲蓄投資差不變，順差就不變。貿易戰打關稅，是花力氣打無效的地方。

**場景三：外國爆發動盪，熱錢湧入台灣**

外資大量進入 → NCO 大幅下降 → 可貸資金供給增加 → 台灣利率被壓低 → 台幣需求暴增 → 台幣升值 → 出口變貴 → NX 下降

> **結論：資本流入的代價是貿易逆差。** 錢進來，最終要買東西，否則留著幹嘛。

## 台灣長期順差的根源

套用公式 **NX = S - I**：

- 台灣儲蓄率長期居全球前幾名（家庭保守、企業留盈餘）
- 國內投資機會有限（市場小，大廠優先海外擴張）
- S 遠大於 I，差額只能變成 NCO 流出，帶動 NX 順差

要縮小台灣的貿易順差，只有三條路：
1. 叫大家少存錢多花費（難）
2. 政府多舉債赤字（有代價）
3. 讓國內投資機會更多（最根本，但最慢）

## 小知識：NX = NCO 在現實中的三個複雜性

這個恆等式作為**會計定義**永遠成立，但現實中有幾個地方讓它看起來沒那麼乾淨：

**①央行外匯存底的角色**
台灣央行持有超過 5,000 億美元外匯存底，累積過程中央行賣台幣買美元，算作官方資本流出——這屬於 NCO 的一部分，但性質和民間對外投資完全不同。如果只看「私人」的 NCO，與 NX 的對應關係就不那麼整齊。

**②美元的儲備貨幣特殊地位（Triffin 困境）**
全球各國需要美元作為儲備和結算貨幣，必須持續買入美元資產（NCO 對美國為負），這讓美國結構性地必須維持貿易逆差來供應美元給世界。這個逆差不完全是「美國人儲蓄率太低」的結果，而是美元體系的設計代價。

**③統計誤差**
理論上全球所有國家的 NX 加總應該等於零（一國的出口是另一國的進口），但實際統計數字每年有數千億美元的「全球經常帳誤差」，代表有資金流動沒被正確計入。所以課本的完美恆等式，在實務數據上只是近似值。
